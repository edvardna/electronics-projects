# 🪑 Parametric 3D Printed Ball Chair (Pallo Tuoli)

A modern reinterpretation of the iconic ball chair, redesigned for additive manufacturing using parametric CAD and modular construction.

---

## 🚀 Overview
This project explores how classic furniture design can be adapted for **3D printing technologies**.

Inspired by the traditional *Pallo Tuoli (Ball Chair)*, the design was reimagined to be:
- Modular  
- Printable  
- Structurally efficient  
- Visually minimal  

The result is a concept that combines **industrial design, engineering, and digital fabrication**.

---

## 🧠 Design Goals
- Translate an iconic form into a **3D printable structure**  
- Optimize geometry for **additive manufacturing constraints**  
- Maintain **ergonomics and visual identity**  
- Enable **modular assembly** for large-scale objects  

---

## 🛠 Tools & Technologies
- **CAD Design Onshape**  
- **3D Printing (FDM technology)**  
- **Parametric modeling techniques**  
- Slicing software (for print preparation)  

---

## ⚙️ Design Approach

### 🔹 Parametric Modeling
The chair was designed using parametric CAD principles, allowing:
- Easy scaling  
- Geometry adjustments  
- Iterative improvements  

---

### 🔹 Modular Construction
Due to print size limitations:
- The chair is split into multiple printable segments  
- Parts are designed for **assembly and alignment**  
- Enables fabrication on standard desktop 3D printers  

---

### 🔹 Structural Considerations
- Wall thickness optimized for strength  
- Curved geometry distributes load efficiently  
- Balance between **weight, strength, and material usage**  

---

### 🔹 Aesthetic Design
- Smooth, continuous surfaces  
- Minimalist form inspired by Finnish design  
- Focus on clean geometry and visual balance
