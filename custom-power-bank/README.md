# 🔋 Custom 18650 Power Bank

A portable energy storage system built using 18650 lithium-ion cells, featuring wired USB output and integrated wireless charging.

---

## 🚀 Overview
This project is a custom-built power bank designed to provide reliable portable power using multiple 18650 lithium-ion cells.

The system includes battery charging, voltage boosting, protection circuitry, and an integrated **wireless charging module**, making it a versatile and practical everyday device.

---

## 🧠 Features
- Multi-cell 18650 battery pack  
- USB 5V output for wired charging  
- Integrated **wireless charging (IKEA module)**  
- Rechargeable system with charging circuit  
- Compact and portable design  

---

## 🛠 Technologies Used
- **18650 Li-ion battery cells (LG)**  
- **TP4056 charging module**  
- **1S BMS (battery protection)**  
- **MT3608 boost converter (3.7V → 5V)**  
- **USB output module**  
- **IKEA wireless charging module**  
- Basic electronics design and wiring  

---

## 🔋 Power System
The system is based on a **multi-cell 18650 lithium-ion battery pack**, providing sufficient capacity for portable applications.

Key elements:
- Safe charging via TP4056 module  
- Battery protection using BMS  
- Boost conversion to stable 5V output  
- Wireless power transfer using integrated charging coil  

This demonstrates practical implementation of **portable power system design**.

---

## ⚙️ System Architecture
- **Battery Pack**
  - Stores energy using multiple 18650 cells

- **Charging Circuit**
  - Controls safe battery charging

- **Boost Converter**
  - Steps up voltage to 5V for USB output

- **Wireless Charging Module**
  - Enables cable-free charging of devices

- **Output Interface**
  - USB port for standard wired charging
